Сайт1 - сайт с лимитом количества лидов в день
Сайт2 - сайт без лимита количества лидов в день.

Заявки направляются поочередно двум сайтам. Если у первого лимитируемого сайта достигнут лимит заявок - заявки направляются только второму сайту.

Каждый день данные о лидах обновляются!

Лимит нужно прописать в файле data.json(поле "limit"), а так же в файле update_data.php в массиве $default_data_array (поле "limit");

Файл leads_limit.php - это скрипт, который должен выполниться для лимитируемого сайта (отправка письма на определенную почту и запись в определенную таблицу);
Файл leads_unlim.php - это скрипт, который должен выполниться для второго, не лимитируемого сайта (отправка письма и запись в таблицу)
